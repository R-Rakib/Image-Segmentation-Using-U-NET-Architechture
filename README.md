## üß† U-Net Architecture

U-Net consists of:
- **Encoder (Contracting Path):** Captures context using convolution and pooling  
- **Decoder (Expanding Path):** Enables precise localization using upsampling  
- **Skip Connections:** Preserve spatial information by combining encoder and decoder features  

This design allows the model to perform **high-precision segmentation even with limited training data**.
